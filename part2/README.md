# HBnB - Part 2: Business Logic & API Implementation

This part focuses on the implementation of the core business logic and the Presentation layer (API) using Flask and Flask-RESTx.

## Key Features
* **User Management**: Endpoints for user registration and retrieval.
* **Place & Review Management**: CRUD operations for places and reviews.
* **Facade Pattern**: Implements the facade pattern to streamline communication between layers.

## Getting Started
1. Navigate to the directory: `cd part2`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python run.py`
