HBnB Evolution
This repository contains the initial design and documentation artifacts for the HBnB Evolution project.

The focus of this repository is on system modeling and architectural planning using UML diagrams before moving into implementation phases.

Project Scope
The repository includes:
UML diagrams describing the system architecture
Domain model representations of core entities
Sequence diagrams illustrating key user and system interactions
Project documentation placeholders

All diagrams are created using Mermaid and stored as source files for easy rendering and maintenance.

Contents
diagrams/
Contains all Mermaid (.mmd) source files for the UML diagrams.

documentation/
Contains project documentation files, including placeholders for future documentation.

Diagrams Included
High-level package diagram
Business logic class diagram
Sequence diagram covering main system scenarios

Notes
This repository represents the design phase of the HBnB Evolution project and serves as a foundation for subsequent development stages.
