# HBnB - Part 3: Authentication & Database

This project implements the backend architecture for the HBnB application, featuring JWT authentication, Role-Based Access Control (RBAC), and persistent database storage using SQLAlchemy and SQLite.

## Project Structure
* **part3/**: Contains the source code for the backend API and database models.

## Requirements
* Python 3.8+
* Flask
* Flask-RESTx
* Flask-JWT-Extended
* SQLAlchemy

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yasir-AL-Shabanah/holbertonschool-hbnb.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd holbertonschool-hbnb/part3
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application:**
   ```bash
   python run.py
   ```

## API Documentation
Once the application is running, you can access the Swagger UI documentation at:
* **URL:** [http://localhost:5000/api/v1/](http://localhost:5000/api/v1/)

Use this interface to test the endpoints (Login, Users, Places, Reviews) directly from your browser.
